# Fashion MNIST Image Classification

This project implements an image classification model using the **Fashion MNIST** dataset. The goal is to classify 28x28 grayscale images of clothing into 10 categories.

## Dataset

The Fashion MNIST dataset contains 70,000 images of fashion items, divided into:

- 60,000 training images
- 10,000 test images

Classes:

1. T-shirt/top
2. Trouser
3. Pullover
4. Dress
5. Coat
6. Sandal
7. Shirt
8. Sneaker
9. Bag
10. Ankle boot

Dataset is built-in in **TensorFlow**.

## Requirements

- Python 3.x  
- TensorFlow  
- NumPy  
- Matplotlib  
- Scikit-learn  
- Seaborn
